Description
===========

Installs jzeromq on Ubuntu systems

Requirements
============
* Ubuntu 10.04 LTS
* Java cookbook
* Zeromq cookbook

Attributes
==========
[:jzeromq][:version]: The hash of the Github commit, which was pulled down
[:jzeromq][:download_url]: The URL of the tarball for jzeromq.  There is no default for this attribute
